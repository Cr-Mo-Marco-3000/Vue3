# Vue/Cli



## 1. Vue/Cli 설치

> vue/cli 설치 및 프로젝트 생성 등은 이전과 동일하다.

- 전체 시스템에 설치할 때
  - `$ npm install -g @vue/cli`
- 해당 폴더에 설치할 때
  - `$ npm install @vue/cli`



## 2. 프로젝트 생성

- 새로운 프로젝트 생성
  - `$ vue create project_name`
  - 해당 폴더로 이동

- Vuex 사용시
  - vue add vuex
- Vue Router 사용시
  - vue add router
